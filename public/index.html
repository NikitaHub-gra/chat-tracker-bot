Завершённых ?
✅
—
Задачи фактически завершённые в текущем месяце.
Просрочено ?
⚠️
—
Активные задачи, открытые > 3 рабочих дней.

Расчёт: рабочие дни (пн–пт) от даты создания до сегодня > 3.

Не попадают: «Отложенные» (даже если старые) — они не активные.

Источник поля: dateTime.dateTimeUtcSeconds из /task/list.

⚠️ SLA-норматив: закрыть или дать ответ за 3 рабочих дня.
Доработка ?
🔧
—
Задачи со статусом «Доработка».

Возвращены на повторное выполнение.
Сегодня ?
📅
— / —
создано / решено
Задачи за сегодняшний день.

Создано — фильтр тип 12 (дата создания) = сегодня.
Решено — два запроса с дедупликацией:
① статус «Завершена» + тип 19 = сегодня
② статус «Доработка» + тип 19 = сегодня (закрытые сегодня, потом переоткрытые)

Совпадает с PlanFix UI «Дата перевода в статус Завершённая».
Нераспределённые ?
👥
—
Задачи без конкретного исполнителя — только группа Service_Support.

Учитываются задачи созданные с 01.01.2026.

Требуют ручного распределения.
Ср. время закрытия ?
⏳
...
Среднее время жизни задач, закрытых в этом месяце.

Фильтр API: тип 19 (дата закрытия) за текущий месяц, статус id=3.
Расчёт: рабочие дни от создания до сегодня.

⚠️ Ограничение точности: вместо даты закрытия используется сегодняшний день — задачи, закрытые в начале месяца, дают завышенный показатель.

Метрика достоверна в середине/конце месяца. В начале месяца — ориентировочная.
Срочные ?📋
🚨
—
Открытые задачи с флажком «Срочная».

Стандартное поле PlanFix — выставляется вручную при создании или в процессе работы.

Типы обращений ?
Разбивка открытых задач по трём полям PlanFix:

🗂️ Тип обращения — Инцидент, Настройка, Консультация и др.
🚗 Выезд — требуется ли выезд специалиста
🏷️ Категория запроса — детальная классификация

Поля только внедряются — у части задач значения ещё не заполнены.

Дежурство ?
Статистика дежурств по сотрудникам за текущий месяц.

📅 Расписание — задаётся вручную в коде (DUTY_SCHEDULE)
📊 Создал — задачи, где дежурный является постановщиком (поставил сам, принимая обращения клиентов)
✅ Закрыл — задачи, где дежурный является исполнителем, закрытые в период смены

⚠️ До 08:00 показывается вечерняя смена предыдущего дня (ночное дежурство).

Кнопка 📅 Расписание — редактировать дежурный график.
